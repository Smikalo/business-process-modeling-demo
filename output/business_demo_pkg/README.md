# Business Demo Package — V12.2_champion

**Date:** 2026-05-07
**Model:** V12.2_champion (production) = `0.925·V11_final + 0.075·V12_external`
**Test window:** Jul 2025 – Jan 2026 (7 months — full held-out period; the model never saw actuals from this window during training)
**Test metrics:** SIMSCORE 0.4435 · WAPE 0.3931 · Bias +2.13 % · Monthly-WAPE 0.0794

## Talking points

1. **На годовом уровне ошибка ~0.6 %** — модель попадает в годовой объём почти точно (31.49 млн факт vs 31.69 млн прогноз = погрешность 200 тыс UAH из 31.5 млн).
2. **На месячном уровне точность ~92 %** (Monthly-WAPE 0.08) — лучше типичной экспертной точности для индустрии (25-35 % WAPE без модели).
3. **На уровне отдельной пары "партнер ? SKU ? месяц" ~63 %** — близко к мировому потолку для подобной задачи (открытые M5 / Rossmann / Favorita плато на 62-67 %).
4. **Декабрьский пик пойман** — самый рискованный месяц года (новогодние подарки) спрогнозирован с ошибкой меньше 1 % по всем 3 брендам.
5. **Январский спад** (после-новогодний) — единственный системный недостаток. Модель завышает на ~24 % из-за того, что в обучающей выборке нормальных январей было мало (война и карантин искажали 2021-2025).

## Что в Excel (`Заказник_AI_прогноз.xlsx`)

Стандартный формат Заказника. На каждый бренд:
* **Brand-level**: Отгрузки Факт (шт), Прогноз AI (шт), Отклонение (%), Отгрузки Факт (тыс UAH), Прогноз AI (тыс UAH).
* **SKU-level**: top-15 артикулов по обороту с Факт + Прогноз AI на каждый из 7 месяцев.

Зелёным фоном — прогноз AI; белым — факт.
